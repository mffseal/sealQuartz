# 间接访问操作符

通过一个指针访问它所指向的地址的过程称为间接访问/解引用指针。

`*`操作符是间接访问操作符。与[[3-计算机科学/6-应用开发/0-软件语言/cpp/指针|指针]]一起使用，用于访问指针所指向的值。该操作符需要一个右值（类型是指针类型表达式），返回一个左值。

多个`*`操作符具有从右向左结合的特性：

```c
int **c = &b;
```

等价于：

```c
int *(*c) = &b;
```

## 示例

表示表达式 `*a`的类型是int。

```c
int *a;
```
