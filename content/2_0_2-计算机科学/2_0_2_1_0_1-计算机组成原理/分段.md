# 分段

分页是针对程序内容的：
**分段存储管理能反映程序的逻辑结构并有利于段的共享**

程序按内存物理地址直接加载，内存的空间利用率不高，因为程序中的各个部分在执行时只有部分在发挥作用，并且程序能直接访问内存，安全性不高。

因此将程序按自身的组织结构进行划分，例如划分出代码段、数据段等。

在分段中，CPU产生一个包含段号和段偏移量的逻辑地址，如果段偏移量比`limits`小，则该地址为有效地址，否则会因为地址无效而引发计算错误。
