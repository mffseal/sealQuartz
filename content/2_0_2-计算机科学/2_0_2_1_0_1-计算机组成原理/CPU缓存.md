# CPU缓存

![[z-oblib/z2-attachments/Pasted image 20220602143516.png]]

| 从 cpu 到 | 大约需要的时钟周期                 |
|------------|-----------------------|
| 寄存器 | 1 cycle (4GHz 的 CPU 约为0.25ns) |
| L1| 3~4 cycle                      |
| L2| 10~20 cycle                    |
| L3| 40~45 cycle                    |
| 内存| 120~240 cycle                  |
