# 访问控制符

![[z-oblib/z2-attachments/v2-315cda6274202d57b75be7641eba442f_1440w.png]]

## public

- 可以被所有其它类访问。

## private

- 只能被自己访问。

## protected

- 只能被自身、子类、同属一个包的其他类访问。
	- 在同一个包相当于public。

## default

- 默认修饰符（friendly）。
- 只能被自身、同属一个包的其他类访问。
