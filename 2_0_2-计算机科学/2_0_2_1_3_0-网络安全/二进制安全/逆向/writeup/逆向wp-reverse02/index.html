# 定位函数

程序时一个 win32 的窗口程序, 按钮是灰色的.
老套路, 根据字符串来定位函数.

这题可以找到一个内容是 flag:{NSCTF_md57e0cad17016b0>?45?f7c>0>4a>1c3a0} 的字符串, 直接提交错误, 看来没这么简单.
