class BigRoom {
 
    public void sleep() {
        synchronized (this) {
            log.debug("sleeping 2 小时");
            Sleeper.sleep(2);
        }
    }
 
    public void study() {
        synchronized (this) {
            log.debug("study 1 小时");
            Sleeper.sleep(1);
        }
    }
 
}

class BigRoom {
 
    private final Object studyRoom = new Object();
    private final Object bedRoom = new Object();
 
    public void sleep() {
    synchronized (bedRoom) {
            log.debug("sleeping 2 小时");
            Sleeper.sleep(2);
        }
    }
 
    public void study() {
        synchronized (studyRoom) {
            log.debug("study 1 小时");
            Sleeper.sleep(1);
        }
    }
 
}

对独立的功能使用独立的多把锁，提高并发度。

注意，给方法加锁还是锁在this上。

# 优缺点

好处，是可以增强并发度 坏处，如果一个线程需要同时获得多把锁，就容易发生死锁
