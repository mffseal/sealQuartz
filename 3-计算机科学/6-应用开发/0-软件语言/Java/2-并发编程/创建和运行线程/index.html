# 方法3 Thread配合FutureTask

FutureTask需要用Callable进行可以处理返回值。

FutureTask<Integer> task = new FutureTask<>(new Callable<Integer>() {  
    @Override  
    public Integer call() throws Exception {  
        log.debug("futuretask");  
        return 123;  
    }  
});  
Thread t3 = new Thread(task);  
t3.start();
