# 指针

指针变量本身就是一个数字，不存在内建的间接访问属性。除非显示的使用间接访问操作符。

# 未初始化和非法的指针

1
2
3

int *a;
...
*a = 12;

a本身就是一个数字：

  • a是一个自动变量，意味着未对其进行初始化前其值是垃圾。
  • a是一个静态变量，意味着未显示初始化情况下其值始终为0。

# 指针的指针

1
2
3

int a = 12;
int *b = &a;
int **c = &b;

# 指针表达式

涉及到变量的地址与值。

1
2

char ch = 'a';
char *cp = &ch;

初始化如下：

# ch

左值：表示表达式值为’a'

右值：表达式为内存的地址（而不是地址上存的值）：

# &ch

右值：变量ch的地址，但是&ch本身的存储位置不确定，可能是栈上，可能在寄存器里，所以浮空表示。 左值：&ch位置不确定，或者说该变量没有对应的地址信息，无法作为左值使用。

# cp

右值：cp是一个指针变量，其值记录的是ch的地址。 左值：cp自己的地址。因为该表达式没有进行间接访问，所以没有箭头。

# &cp

右值：取cp的地址值，但是取出来不确定放在哪里。 左值：&cp没有自身的地址信息，无法作为左值。

# *cp

右值：进行了一次间接访问，跳转到了ch的值 左值：进行了一次间接访问，跳转到了ch的地址

# *cp+1

右值：表达式最终结果存储位置未知，非法右值。 左值：取ch的地址值并+1。

# *(cp+1)

右值：指针运算结果为右值。 左值：*操作符将指针运算cp+1的结果（一个常量）转为了左值。

# ++cp

表达式结果返回的是一个右值，同时会更新掉cp的值。先获取cp值拷贝并增加值大小，再返回。

# cp++

表达式结果返回的是一个右值，同时会更新掉cp的值。先返回cp值拷贝，再增加cp的值。

# *++p

间接访问操作符作用于增值后的指针拷贝：

# *cp++

后缀++操作符优先级高于*操作符：

  1. ++操作符产生cp的一份拷贝
  2. ++操作符增加cp的值
  3. cp的拷贝上执行间接访问

该表达式常出现在循环体内用于一次访问数组内容。

# ++*p

首先执行间接访问，再将cp指向位置上的值增加1，表达式结果是此时该值的一个拷贝。

# (*cp)++

与++*p一样。

# ++*++cp

  1. 执行++cp，返回增加后的cp的值的拷贝。
  2. 对拷贝值进行左++操作。

# ++*cp++

  1. 执行cp++，返回原cp值的拷贝值。
  2. 执行对拷贝值的间接访问，效果等同于*cp
  3. 对cp所指向位置的值进行左++，返回增加后的拷贝值。
